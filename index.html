Prework Study Guide

HTML

  • The head element contain information about webpage
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The title element defines the title of the webpage, it is shown in the browser`s title bar
  • The body element represents the visible content shown to the user
  • The div element creates a division in the page
  • The span element creates an inline division, used to mark up a part of a text
  • The br element inserts a single line break.
  • The id attribute a way to identify elements and select them

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • * is wild card symbol
  • The rel attribute specifies the relationship between the current document and the linked document resource
  • The href attribute specifies the location (URL) of the external resource
  • The class attributes allow us to share a CSS rule with any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • cd: navigate into a directory
  • code .: open VS Code
  • git clone: clone the SSH value
  • git status: checks what branch we are currently on
  • git checkout: switches to a branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all changes to next commit
  • git commit -m "": commit changes to local branch
  • git pull origin main: double-check if our local branch is in sync with the base branch in GitHub
  • git push origin branch name: push local branch to the remote branch in GitHub
